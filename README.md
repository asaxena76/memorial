# Memorial

A private memorial space built with Next.js + Firebase. Family members sign in, await approval, and share memories with moderation.

## Requirements
- Node.js 20+
- Firebase project with Auth, Firestore, Storage, and Functions enabled

## Local dev
```
npm install
npm run dev
```

Create `.env.local` with Firebase web config keys (see `docs/FIREBASE_SETUP.md`).

## Firebase setup
See `docs/FIREBASE_SETUP.md` for required configuration, CORS, and admin bootstrap steps.
